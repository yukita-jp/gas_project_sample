# gas_project_sample

## デプロイ
### 開発環境にデプロイ
$ npm run deploy

### 本番環境にデプロイ
$ npm run deploy:prod

*開発・本番は現時点では同じ設定。切り替えが必要な場合は、clasp-dev.jsonとclasp-prod.jsonの内容を変更。
